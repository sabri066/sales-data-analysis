# 📊 Sales Data Analysis Using Python

## 📌 Objective
Analyze sales data to identify patterns, trends, and business insights.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 🔄 Process
- Created and structured dataset
- Calculated total sales
- Performed data grouping (Product & Region)
- Visualized results using charts
- Extracted meaningful insights

## 📊 Key Insights
- Identified top-performing product
- Found best-performing region
- Analyzed average sales trends

## 📁 Files
- sales-data-analysis.ipynb

## 🚀 Outcome
This project demonstrates how raw data can be transformed into actionable business insights.
